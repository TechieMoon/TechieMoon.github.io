Featured Project 03

Real / Fake Voice Classifier with wav2vec2

Transformer 기반 음성 진위 분류 실험

수업 프로젝트
Repository

인공지능이 생성한 가짜 음성과 사람이 만든 진짜 음성을 분류하기 위해 wav2vec 2.0 base 모델을 파인튜닝한 프로젝트입니다. 단순 학습뿐 아니라 LoRA rank, data augmentation, pruning 전후를 비교하며 실험을 구조화했습니다.

  • LoRA rank / augmentation / pruning 비교 실험 수행
  • 포트폴리오 기준 최고 validation accuracy 0.9976 기록
Python PyTorch Hugging Face wav2vec2 LoRA Colab
Featured Project 04

Character Face Classification

ResNet18 파인튜닝 이미지 분류

개인 프로젝트
Repository

애니메이션 캐릭터 8종의 얼굴을 분류하는 딥러닝 프로젝트입니다. 직접 영상 프레임을 추출하고 얼굴 중심으로 crop하여 데이터셋을 구축했으며, ResNet18 transfer learning으로 학습했습니다.

  • 8개 class, class당 120장 기준의 데이터셋을 직접 구축
  • RandomResizedCrop, HorizontalFlip, Normalize 기반 전처리 적용
  • 모델 학습부터 테스트 이미지 예측까지 전체 흐름 구현
Python PyTorch ResNet18 Computer Vision Dataset Curation
